getInstance

fun getInstance(context: Context, enableCrashlytics: Boolean = false): CmSdk

Returns an instance of CmSdk, fully initialized and ready to use. Note: use CmSdk.setApiKey to configure the API key provided by Comodule.

Return

An instance of CmSdk that is ready to be used throughout the application.

Parameters

context

The context required for the SDK's operation, usually your application context.

enableCrashlytics

Optional parameter to enable or disable Crashlytics integration. If set to true, Crashlytics will be initialized and enabled for the SDK instance. Crashlytics collects non-fatal SDK errors and reports them to the Firebase project of the client application. No Crashlytics data is collected by Comodule.

Defaults to false, meaning Crashlytics will not be enabled unless explicitly requested.
